FoxApi

Javascript Library for interacting with FoxCommerce API

Usage example

const api = new FoxApi({api_url: 'http://api.foxcommerce'});
api.products.search({state: 'cart'});

Options

Option Type Default Description
api_url String Required option. Should point to phoenix backend.
version String v1 Protocol version.

Methods

Method Returns Description
getCreditCards() Promise

Base methods

Method Returns Description
addAuth(<String> jwt) FoxApi Set jwt authorization header for next requests
addHeaders(<Object> header) FoxApi Set http headers for next requests
uri(<String> uri) String Prepares and returns final url which will be used in request.
request(<String> method, <String> uri, <Object> data?, <Object> options?) Promise Makes http request, possible options:
  • headers: headers to sent
get(<String> uri, <Object> data?, <Object> options?) Promise Makes GET http request
post(<String> uri, <Object> data?, <Object> options?) Promise Makes POST http request
patch(<String> uri, <Object> data?, <Object> options?) Promise Makes PATCH http request
delete(<String> uri, <Object> data?, <Object> options?) Promise Makes DELETE http request

Addresses methods

Method Returns Description
getAddresses(<Query options> options) Promise<Addresses> Choices for options.sortBy parameter: id regionId name address1 address2 city zip isDefaultShipping phoneNumber deletedAt region_id region_countryId region_name region_abbreviation
getAddress(<Number> addressId) Promise<Address>
addAddress(<CreateAddressPayload> address) Promise<Address> Adds new address
editAddress(<Number> addressId, <UpdateAddressPayload> address) Promise<Address> Updates selected address
setAddressAsDefaut(<Number> addressId) Promise<Address> Sets selected address as default.
removeDefaultAddress() Promise Removes default address.
deleteAddress(<Number> addressId) Promise Deletes address.

Auth methods

Method Returns Description
signup(<String> user, <String> password) Promise Register new user
login(<String> user, <String> password) Promise Authenticate user by username and password, returns promise with jwt token

Cart methods

Method Returns Description
updateQty(<String> itemId, <Number> qty) Promise Updates quantity for selected item in the cart
addToCart(<Array> items) Promise Add new items to the cart
removeFromCart(<Array> items) Promise Remove items from the cart
emptyCart() Promise Erase all items from the cart
